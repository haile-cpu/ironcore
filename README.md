# IronCore Fitness Landing Page

A modern, professional landing page for IronCore Fitness, designed to showcase gym services, training programs, and community features. Built with clean HTML, CSS, and JavaScript for optimal performance and responsiveness.

## 🚀 Features

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Hero Section**: Eye-catching introduction with compelling call-to-action buttons
- **Services Showcase**: Highlighted gym services including coaching, cardio, weight training, and recovery
- **Training Programs**: Detailed program cards with interactive discovery features
- **Coach Profiles**: Meet the expert coaching team with professional photos and bios
- **Pricing Plans**: Clear pricing tiers with feature comparisons and plan discovery buttons
- **Blog Section**: Featured articles and blog previews with links to full posts
- **Contact Information**: Complete contact details including address, email, and phone
- **Interactive Elements**: Hamburger mobile menu, smooth scrolling, and plan highlighting
- **Modern UI**: Clean design with professional typography and color scheme

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for accessibility and SEO
- **CSS3**: Custom properties, Flexbox, Grid, and responsive design
- **JavaScript**: DOM manipulation for interactive features
- **FontAwesome**: Icon library for UI elements

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. **Clone the repository** (if applicable) or download the project files
2. **Navigate to the project directory**
3. **Open `index.html` in your web browser**

That's it! The website is static and requires no build process or server setup.

## 📱 Usage

- **Navigation**: Use the header menu to jump to different sections
- **Mobile Menu**: Click the hamburger icon on mobile devices
- **Plan Discovery**: Click "Discover Plan" buttons to highlight relevant training programs
- **Blog**: Access articles from the blog section or dedicated blog page
- **Contact**: Find contact information in the footer

## 🎨 Customization

### Colors

The design uses CSS custom properties for easy theming:

- `--accent`: Primary brand color (#d7ff1b)
- `--dark`: Dark background (#070f22)
- `--text`: Primary text color (#f6f9ff)

### Content

Update text, images, and contact information directly in the HTML files.

### Images

Replace Unsplash URLs with your own images while maintaining the same structure.
